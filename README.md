# notes_cms
Dự án là một nền tảng quản lý công việc và tri thức dạng Notion, cho phép người dùng tạo workspace, quản lý trang (page) và khối nội dung (block) linh hoạt, hỗ trợ database đa dạng (table, board, calendar, timeline), chỉnh sửa realtime nhiều người, phân quyền chi tiết, chia sẻ và xuất bản nội dung.
